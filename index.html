An example of task solution

Task: function should return true if the input number is narcisstic. A Narcissistic Number is a positive number which is the sum of its own digits, each raised to the power of the number of digits in a given base.

Solution:

          

  function narcissistic (value) {
    const valueString = value.toString();
    let sum = 0;
    for (let i = 0; i < valueString.length; i++) {
      sum += Math.pow(valueString[i], valueString.length);
    }
    if(parseInt(sum) === value) {
      return true;
    }
    return false;
  }
